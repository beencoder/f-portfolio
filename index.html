Let's Play!
  • 01

    Music Player

    javascript로 제작한 음악 플레이어입니다.
    화면에 노래 재생과 정지, 이전 노래, 다음 노래, 재생바, 루프 기능 그리고 노래 목록 화면까지 구현하였습니다.
    플레이 버튼을 눌러 Coldplay의 노래를 감상해보세요!
  • 02

    Tetris Game

    javascript로 제작한 테트리스 게임입니다.
    방향키로 블록을 쌓고 한 줄이 채워지면 점수를 얻고 맨 위까지 쌓일 시 게임 오버가 되도록 구현하였습니다.
    플레이 버튼을 눌러 추억의 테트리스 게임을 해보세요!
  • 03

    Memory Game

    javascript로 제작한 기억력 게임입니다.
    카드를 클릭하며 뒤집어진 카드들의 위치를 기억하고 모든 쌍을 찾으면 게임이 끝나도록 구현하였습니다.
    플레이 버튼을 눌러 기억력이 좋은지 확인해보세요!
  • 04

    Typing Game

    javascript로 제작한 타이핑 게임입니다.
    시작 버튼을 눌러 화면에 보이는 영단어를 6초 안으로 오타 없이 타이핑해야만 점수를 얻도록 구현하였습니다.
    플레이 버튼을 눌러 업무 전 손풀기 운동을 해보세요!
